# Changelog

All notable changes to V-Mail will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/), and this project
adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.5.0] - 2026-03-06

### Added

- Advanced Search with Lunr.js integration
- Contacts Integration
- Calendar Integration
- Email Signatures Feature
- File Attachments Preview
- Comprehensive tests for TaskManagement hook
- Tests for Email Templates

### Fixed

- TypeScript compilation errors across multiple components and hooks (PR #67)
- Test type errors in hooks and services (PR #66)
- All test failures resolved (PR #68)
- Linter errors in App.tsx (PR #69)
- ESLint warnings in AdminPanel, CacheSettings, Calendar, ComposeModal, DelegationSettings (PR #71)

### Changed

- CI/CD pipeline simplified and optimized
- Snyk security scanning configuration enhanced
- Code coverage job added to CI pipeline

### Maintenance

- Repository cleanup: removed legacy files (js/, css/, old configs)
- Synchronized version numbers across all sub-projects
- Fixed lint-staged configuration to include TypeScript files
- Cleaned up temporary PR description files
- Consolidated duplicate test configurations
- Updated index.html to proper Vite SPA entry point
