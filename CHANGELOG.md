# Changelog

All notable changes to V-Mail will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.5.0] - 2026-03-06

### Added
- Advanced Search with Lunr.js integration
- Contacts Integration
- Calendar Integration
- Email Signatures Feature
- File Attachments Preview
- Comprehensive tests for TaskManagement hook
- Tests for Email Templates

### Fixed
- TypeScript compilation errors across multiple components and hooks

## [1.4.0] - 2026-03-06
